React Date Ranger

This components lets you pick date ranges using common periods used in reporting: year, quarter, month, week and day.

Component props

allowedRange Array of two strings representing dates. Date picker will be limited by these dates.
allowedRangeFormats String format for moment.js date parser which will be used to parse allowedRange prop. Default is 'YYYY-MM-DD'.
allowedPeriods String containing all allowed periods to chose from. Default is 'YQMWD'
period Preset period. Default is first from of allowed periods.
range Preset selected range. Default is start and end of preset/default period.
onChange Change callback is passed `range` parameter. Which is an array of two moment.js objects.
